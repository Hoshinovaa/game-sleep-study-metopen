# Gaming Duration and Sleep Time Analysis

This repository contains a LaTeX research paper and dataset used to analyze the relationship between gaming duration and student sleep time.

## Structure

paper/            -> LaTeX source files for the research paper  
data/             -> Dataset used in the study (Kaggle dataset)  
figures/          -> Figures and visualizations used in the paper  
image_rev/        -> Revised figures used in the final version of the paper  
latex-template/   -> LaTeX class, style, and bibliography style files (.cls, .sty, .bst)  
Definitions/      -> Additional template definitions and journal configuration files
